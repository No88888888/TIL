# 알고리즘1

무엇이 좋은 알고리즘인가?

1. 정확성
2. 작업량 - 시간복잡도로 표현한다
   1. 시간복잡도
      - 실제 걸리는 시간을 측정
      - 실행되는 명령문의 개수를 계산
3. 메모리 사용량
4. 단순성
5. 최적성

### 배열

일정한 자료형의 변수들을 하나의 이름으로 열거하여 사용하는 자료구조

### 정렬

2개 이상의 자료를 특정 기준에 의해 작은 값부터 큰 값 혹은 그 반대의 순서대로 재배열 하는 것

- 버블정렬
  - 인접한 두 개의 원소를 비교하며 자리를 계속 교환하는 방식
- 카운팅 정렬
  - 항목들의 순서를 결정하기 위해 집합에 각 항목이 몇 개씩 있는지 세는 작업을 하여, 선형 시간에 정렬하는 효율적인 알고리즘
